Introducción a Html

Ideas principales

    El html es el componente más básico de la web, este se utiliza para definir y crear la estructura del contenido web.

    Utiliza diferentes tecnologías como CSS para dar una mejor apariencia y presentación.

    Html utiliza elementos para darle estructura a la página web, cada una con sus diferentes funciones.

    La gran mayoría de las etiquetas del Html debe tener un inicio y un fin.

    El inicio de una etiqueta es normal es decir; lleva los dos carácteres y dentro de estos se encuentra el nombre.

    Para el cierre de la etiqueta, antes de escribir el nombre de la etiqueta se escribe una diagonal.

    Entre el inicio y fin de la etiqueta debe estar el contenido.

Apuntes

Tim Berners Lee desarrolló el html

El html es un lenguaje de marcado para realizar la estructura del contenido web.

Este tipo de lenguaje se ejecuta en un servidor y se tarduce para el usuario.

Se utilizan diferentes elementos para ejecutar la estructura del contenido web, como head,body,footer entre otros.

Html utiliza caracteres de menor que (<) y mayor que (>) para etiquetar el contenido para mostrarlo en un navegador web o que aparezca en la interfaz.

Resumen

El html (Hipertexto de lenguaje de marcado) es un lenguaje de marcado básico de la web
que permite ejecutar la estructura de su contenido y
utiliza elementos con diferentes funciones con su respectivas etiquetas de inicio y fin
(aunque hay algunos que no tienen ambas) para realizarlo,
asi como también utiliza lenguajes como Css, Javascript entre otros,
para dar una mejor apariencia y vizualización para los usuarios.

Texto en Html

Ideas principales

    El texto en el HTML se estructura de forma más simple para que el navegador lo muestre correctamente.

    El contenido estructurado simplifica la experiencia en la lectura y se disfruta más.

    La mayor parte del texto estructurado está compuesto por encabezados y párrafos.

    En html hay seis elementos de encabezado:h1,h2,h3,h4,h5 y h6 cada uno con sus respectivas etiquetas. (<>)

    En html se utiliza el elemento < p> , /< p> para hacer un párrafo.

    Se utiliza el elemento listas ordenadas (ol) para realizar listas que tienen un determinado orden.

    También se utiliza el elemento listas desordenadas (ul) para realizar listas que no tienen un determinado orden.

Apuntes

Una de las principales funciones de Html es dar al texto estructura y significado de forma que un navegador pueda mostrarlo correctamente.

Es mejor usar solo un h1, por página ya que es el nivel del titulo superior y todos los demás se sitúan por debajo de él.

Html utiliza diferentes elementos para dar enfásis e importancia a los textos como:
< b>: Se utiliza para poner el texto en negritas
< strong>: Se utiliza para poner el texto en negritas.
< i>: Se utiliza para poner el texto itálica.
< em>: Se utiliza para un texto con énfasis.
< mark>: Se utiliza para un texto marcado.
< small>: Se utiliza para hacer un texto más pequeño
< del>: Se utiliza para >tachar el texto
< sub>: Se utiliza para un texto subíndice
< sup>: Se utiliza para un texto superíndice

Resumen

El html utiliza elementos como encabezados (h1,h2,h3,h4,h5,h6),
párrafos (p), listas ordenadas (ol), listas desordenadas (ul)
y elementos de enfásis para estructurar un texto simple para mejorar la lectura del
usuario al navegar en la interfaz de la página.

Imagenes en Html

Ideas principales

    Para añadir imágenes a un documento html se utliza el elemento img

    El elemento img tiene diferentes atributos como src="", este indica la ruta de la imagen que se quiere mostrar.

    Utliza atributos como width y height que son el ancho y la altura que quieres que ocupe la imagen en la página.

    Tambien utliza el atributo alt que especifica un texto alternativo que se muestra en vez de la imagen.

    Y por último se utiliza el atributo tittle para dar nombre a la imagen.

Apuntes

Las imágenes en la web se utlizan no solo por cuestiones estéticas sino para ayudar
al usuario a tener una conexión gráfica con lo que leen en el navegador.

En el html las imágenes se guardan en una carpeta dentro de la carpeta donde se realiza el proyecto web.

Para poder vincular el documento html y la carpeta de imágenes se escribe la ruta de la siguiente forma:
< img src="carpeta/imagen">

Para vincular una imagen de otra página web se escribe la ruta de la siguiente forma:
< img src="http://web.com/carpeta/archivo.jpg">

Resumen

Las imágenes en la web hacen que los usuarios tengan una conexión visual
con lo que leen en el navegador, asi como también porpociona un descanso en su lectura.
Para guardar imagenes en html se crea una carpeta dentro de la carpeta del proyecto web y para vincular uno con otro se utiliza la etiqueta img y dentro de ella se pone el nombre de la carpeta y la imagen que quieras colocar, también se puede vincular una imagen de otra página utilizando la misma etiqueta pero en este caso poniendo el link de esa imagen.

Enlaces en Html

Ideas principales

    La web funciona mediante links, que nos llevan a otras páginas,otros sitios web,
    imágenes, escribir un email entre otras cosas.

    Un link puede ser cualquier cosa
    por ejemplo un texto, una imagen, un botón, etc.

    Se utiliza el elemento a para definir el link.

    En el mismo elemento a se utliza el atributo
    href="" para definir el destino del link.

Por defecto los enlaces en html aparecerán de la siguiente manera:

    Un enlace no visitado estará subrayado y azul
    Un enlace ya visitado estará subrayado y morado
    Un enlace activo esta subrayado y rojo

Apuntes

Se puede usar un imagen como link, simplemente poniendo la etiqueta img dentro de la etiqueta a

En cada link se utiliza un atributo target para especificar en donde abrir el documento vinculado y puede tener los siguientes valores:

    _self:Este esta predeterminado, permite abrir el documento en la misma ventana o pestaña en el que se hizo click.
    _blank:Abre el documento en una nueva ventana o pestaña.
    _parent:Abre el documento en el marco primario.
    _top:Abre el documento en el cuerpo completo de la ventana.

Resumen

Los links son elementos base para la web, ya que permite al usuario navegar la página de forma sencilla,
en html se sigue una sintaxis para colocar links utilizando en primer lugar la etiqueta a y dentro de este se define el destino, se nombra el destino y se pone el nombre del link que aparecerá en el navegador.

Introducción a Css

Ideas principales

    El CSS (Hojas de estilo en cascada) es un lenguaje que define la apariencia de un documento html.

    Son un mecanismo simple para agregar estilo
    o diseño por ejemplo, fuentes, colores, espaciado a documentos web.

    Para agregar el CSS al html exiten tres formas:

      Hoja de estilo incrustada,hojas de estilo en linea y hoja de estilo externa.

      Cada hoja de estilo tiene diferentes funciones y sus respectivas ventajas y desventajas.

      En una hoja de estilo incrustada se utiliza la etiqueta style y se coloca en el head, este se puede usarse en un documento de html único.

      En una hoja de estilo en linea se utiliza la etiqueta style pero este puede colocarse en cualquier elemento del html, por ejemplo body.

      En una hoja de estilo externa es enlazada a un documento externo del html, mediante el elemento link de html y se coloca en el head.

Apuntes

En css se utilizan propiedades para dar un estilo a cada diferente elemento como por ejemplo; margin y padding el primero permite definir un margen externo a cierto elemento y el segundo define un margen interno y a este se le suma el valor
del margen externo por lo que se extenderá el elemento.

Tanto margin como padding permite definir el valor del top,left,right y bottom del el elemento que se le quiere dar un estilo.

También se utilizan propiedades como border, color, font,widht, height, entre otros.

Resumen

El CSS es un tipo de lenguaje que permite definir el estilo de un documento html,
para hacer esto se utilizan hojas de estilo en donde en ellas se usan propiedades básicas
y en cada una de ellas se define un valor en este caso en pixeles o porcentajes.

Proceso de diseño

Ideas principales

    Es una serie común de pasos y fases para el desarrollo de un diseño, en este caso web.

    Se debe centrar el diseño web en sus usuarios y no en preferencias propias.

    Esta serie de fases tienen cáracter iterativo.

    La primera fase es la investigación y evaluación de las preferencias del usuario.

    La segunda es la planificación de la idea del diseño en base a esas preferencias.

    Y por último el desarrollo del diseño.

Apuntes

Para planificar el diseño web el diseñador necesita obtener información precisa tanto de las necesidades y objetivos del proveedor como del usuario. se puede obtener esta clase de información mediante entrevistas o evaluaciones.

La etapa de planificación, se trata de recoger toda la información necesaria posible para hacer tu diseño agradable para los usuarios.

Resumen

El proceso de diseño centrado en el usuario es el conjunto de pasos o fases que todo diseñador debe de seguir, en esto el diseñador investigará y recogerá información necesaria con diferentes métodos de evaluación para poder diseñar en base a las preferencias y necesidades del usuario objetivo.
